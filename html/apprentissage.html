apprentissage - cours r3.03 : html/css, javascript

Présentation

Pour mener à bien la réalisation de ce site web personnel, il est essentiel d'acquérir les bases du HTML qui permet de structurer le contenu d'une page web, du CSS qui assure une mise en forme visuelle de manière claire et attrayante, ainsi que du JavaScript qui rend le site web interactif et dynamique. Ces trois compétences constituent les fondations indispensables du développement web et permettent de transformer une simple idée en un site complet et fonctionnel.

Dans cette page, je vous présente ce que j’ai appris tout au long du cours R3.03 - HTML/CSS, JavaScript.

Connaissances apprises du cours R3.03

i. html

1. Introduction au HTML

  • Le protocol du HTTP : C'est un protocol de communication entre le client (le navigateur) et le serveur. Le navigateur envoie une requête au serveur pour demander une ressource (des pages web en HTML). Le serveur répond ensuite avec le contenu demandé et un code d'état - par exemple : 200 → OK, 404 → Not Found, 500 → Internal Server Error, etc.
  • L'URI du protocole HTTP : Une URI est une chaîne de caractères qui identifie par son ou sa localisation une ressource. On parle plus spécifiquement d'URL (Uniform Ressource Locator).
  • HTML (Hyper Text Markup Language) : Un langage des balises qui permet de structurer le contenu d'une page web.

2. Écriture des balises en HTML

  • 2.1 Système balise ouvrante/fermante : C'est un moyen de délimiter le début et la fin d'une mise en forme à appliquer à du texte. La mise en forme s'applique sur le contenu qui est situé entre la balise ouvrante et la balise fermante.

    Exemple : <b>du texte qui sera en gras</b> → le texte situé entre <b> et </b> sera affiché en gras

  • 2.2 Système balise ponctuelle : C'est un moyen d'indiquer une action de mise en page dans le document (qui ne soit pas une mise en forme d'un contenu précis, mais une action précise plutôt)

    Exemple : <br/> → effectue un saut de ligne (cela ne s'applique pas à un contenu de texte, mais agit pour effectuer l'action de saut de ligne dans le document.)

  • 2.3 Structure d'un fichier HTML : Voici la structure de base d'un fichier HTML vide


  • 2.4 Balises HTML : Il est important de connaître les principes balises HTML de base pour pouvoir écrire correctement une page web. Voici quelques exemples de balises fondamentales
    • Balises de titre HTML : <h1></h1> jusqu'à <h6></h6> → Pour définir les titres et sous-titres en mettant en gras et avec saut de ligne avant et après
      • <h1></h1> : Titre principal de la page
      • <h2></h2> : Sous-titre ou titre de section
      • <h3></h3> : sous-titre d'un <h2>
      • etc.
    • Balises du format du texte → Pour formater du texte
      • <b></b> : texte en gras
      • <i></i> : texte en italique
      • <u></u> : texte souligné
      • <s></s> : texte barré
      • <p></p> : paragraphe
      • etc.

  • 2.5 Balises avec attributs : Toutes les balises peuvent avoir des attributs. Un attribut est un paramètre supplémentaire qui s'ajoute au comportement de base d'une balise pour le modifier et le compléter.


  • 2.6 Balises importantes
    • Lien : <a></a> → Création de liens hypertextes
      Exemple : <a href="https://www.google.com/" target="_blank">du texte à cliquer pour ouvrir le lien nouvelle fenêtre</a>

    • Image : <img/> → Insérer une image à l'address indiquée
      Exemple : <img src="http://www.truc.org/image.jpg" width="50px" />

    • Listes : Deux types de listes : liste ordonnée et liste non ordonnée
      • Liste non ordonnée (à puces) : <ul> <li></li> </ul>
      • Liste ordonnée (à numéros) : <ol> <li></li> </ol>

    • Tableaux : Création d'un tableau
      • <table></table> : tableau
      • <caption></caption> : titre du tableau
      • <tr></tr> : ligne du tableau
      • <th></th> : cellule de légende de ligne ou de colonne
      • <td></td> : cellule du tableau

  • 2.7 Élément link : <link/> → La balise link permet de spécifier les URLs de fichiers liés d'une manière ou d'une autre au document.
    Exemple : <link rel="stylesheet" type="text/css" href="cours.css" />

  • 2.8 Balises meta : <meta/> → Les balises META servent à placer des métadonnées dans une page HTML. On placera ces balises dans l'élément "head" et elles ne seront pas affichées sur la page.
    Exemple : <meta charset="UTF-8" />

ii. css

1. Introduction au CSS

  • CSS (Cascading Style Sheets) : Un langage de mise en forme des documents HTML et XML.
  • Trois façons d'utiliser le CSS
    1. CSS en ligne (inline) - directement dans une balise HTML avec l'attribut "style"
      Exemple : <p style="color: blue;">Texte en bleu</p>

    2. CSS interne - dans la section <head> du fichier HTML
      Exemple : <head> <style> p {color: blue;} </style> </head>

    3. CSS externe - dans un fichier séparé (style.css)
      Exemple : <link rel="stylesheet" href="style.css" />

2. Philosophie du CSS

On considère que chaque contenu encadré par une balise du HTML est un conteneur qui contient un contenu. Un conteneur est une boîte rectangulaire dans laquelle se place le contenu. On parle du modèle de boîte.

  • margin : une marge autour à l'extérieur
  • padding : une marge autour à l'intérieur
  • border : le contour par un rectangle

3. Écrire du CSS

Le code CSS est toujours structure de la même façon ci-dessous :

  • sélecteur : indique à quel élement appliquer le style. Par exemple: p, .classe, #id, etc.
  • propriété : indique ce qui sera modifié dans la mise en forme de la balise. Par exemple: color, font-size, margin, background-color, etc.
  • valeur : la nouvelle valeur qui permet de changer la propriété. Par exemple: red, 16px, center, left, etc.

4. Display

La propriété CSS display est la plus importante pour contrôler la mise en page.
  • Display: "block" → Un élément block-level commence sur une nouvelle ligne et s'étire autant qu'il peut sur la gauche et la droite.
    Des éléments block-level : <div> - <p> - <hr> - <table> - etc.
  • Display: "inline" → Un élément inline peut encadrer du texte dans un paragraphe sans interrompre le flux de ce paragraphe.
    Des éléments inline : <span> - <a> - <b> - <i> - etc.
  • Display: "none" → Pour masquer ou afficher facilement des éléments en CSS

6. Propriétés essentielles

  • Textes et polices : font-family, font-weight, font-size, color, text-align, etc.
  • Arrières-plans : background, background-image, background-size, background-position, etc.
  • Listes : list-style, list-style-type, etc.
  • Bordures et contours : border, border-radius, border-color, margin, padding, etc.
  • Positionnement : display, float, position, top, etc.
  • Animations: animation, animation-delay, animation-duration, etc.

iii. javascript

1. Introduction au JavaScript

  • JavaScript : est un langage de script qui sert à rendre les pages webs dynamiques et interactives.
  • Le rôle principal du JavaScript est de réagir aux actions de l'utilisateur, par exemple: clics, saisies, survols de souris, etc. et de modifier le contenu de la page.
  • Comment intégrer le code JavaScript dans le fichier HTML : On écrit le code JavaScript dans une balise suivante : <script type="text/javascript">Code JavaScript ici</script>

2. Les évènements

  • 2.1 DOM (Document Object Model) : DOM est une description structurée de l'ensemble des éléments contenus dans un document web, sous une forme arborescente. C'est une représentation du document HTML d'origine sous une forme appelée plus précisément "arborescence de noeud" ou "nod tree" en anglais.

    Autrement dit, le DOM est considéré comme le pont entre HTML (le contenu) et JavaScript (le comportement). Il permet à votre script d'interagir avec la page web comme s'il manipulait une arborescence d'objets.

  • 2.2 Gestionnaire d'évènements par attributs HTML : Le code JavaScript est directement intégré dans la balise HTML à l'aide d'un attribut.
    Exemple : <button onclick="alert('Bonjour !')">Essai</button>

    Liste des évènements importants:
    • onclick : l'évènement se produit quand l'utilisateur clique sur l'élément
    • onmouseover : permer de savoir que le curseur de la souris survole sur un élément
    • ondblclick : l'évènement se produit lorsque l'utilisateur double clique sur l'élément
    • etc.

  • 2.3 Gestionnaire d'évènement par propriétés HTML : Le gestionnaire d'évènement est défini dans le code JavaScript en attribuant une fonction à une propriété d'évènement de l'élément.
    Exemple :
    Code HTML
    <button id="btn">Cliquer</button>
    Code JavaScript
    var b = document.getElementById("btn");
    b.onclick = function() {alert("Bouton cliqué!")}
  • 2.4 Gestionnaire d'évènement avec EventListener : La méthode addEventListener() permet d'ajouter un ou plusieurs gestionnaires d'évènements.
    Exemple :
    Code HTML
    <button id="btn">Cliquer</button>
    Code JavaScript
    var b = document.getElementById("btn");
    b.addEventListener('click', function() {alert('Bouton cliqué!')})
